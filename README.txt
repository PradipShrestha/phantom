INTRODUCTION
------------

This module provides an API to the PhantomJS is a headless WebKit scriptable binary
(http://phantomjs.org/).  PhantomJS has a  JavaScript API and fast and native support for 
various web standards: DOM handling, CSS selector, JSON, Canvas, and SVG.  


INSTALLATION
------------

1.  Download and unzip the file (or checkout from github) into one of the valid modules 
directories in your Drupal installation (e.g.:  sites/all/modules).

2.  IMPORTANT:  You must download a binary of PhantomJS and place it in the phantom/bin
directory.  You can find the appropriate binary here: 

http://phantomjs.org/download.html

Note:  You can build PhantomJS from source, but it is time-consuming, dependency-heavy,
and not recommended by the PhantomJS maintainers.

3.  Enable the module.
